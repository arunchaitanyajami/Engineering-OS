# Development Setup

## Requirements

- Node.js 20+
- pnpm 10+
- Rust toolchain for Tauri development
- system dependencies required by Tauri on your operating system

## Install

```bash
pnpm install
```

## Validate The Repository

```bash
pnpm check
```

## Start The Desktop Shell

```bash
pnpm dev
```

## Start The Browser Shell Only

```bash
pnpm dev:web
```

## Run The CLI

```bash
pnpm dev:cli
```
