# Testing

## Test Stack

- Vitest for unit and integration tests
- Playwright for end-to-end smoke coverage

## Commands

- `pnpm test:unit`
- `pnpm test:integration`
- `pnpm test:e2e`
- `pnpm test`

## Expectations

- add focused tests for contracts, validation, migrations, and boundaries
- avoid low-value tests that only repeat implementation details
- keep the desktop smoke test stable and fast during Milestone 0
