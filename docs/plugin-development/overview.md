# Plugin Development Overview

## Intent

Plugins are the only supported way to integrate external systems into Engineering OS.

## Requirements

- independently installable
- independently removable
- permission aware
- testable in isolation
- MCP compatible where possible

## Planned Building Blocks

- plugin manifest
- plugin permissions model
- plugin loader
- tool, resource, and prompt registration
- enable and disable lifecycle

## Reference Path

The first reference connector is the filesystem plugin planned for Milestone 2.
