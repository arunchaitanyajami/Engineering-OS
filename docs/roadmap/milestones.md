# Roadmap

## Milestones

1. `M0` Foundation and Project Constitution
2. `M1` Desktop Shell and Core Runtime
3. `M2` Plugin SDK and MCP Gateway
4. `M3` GitHub Connector and PR Review Agent
5. `M4` Jira Connector and Ticket Analyst Agent
6. `M5` Confluence Connector and Design Doc Agent
7. `M6` Local Memory and Semantic Search
8. `M7` Google and Microsoft Connectors
9. `M8` Voice Assistant
10. `M9` Workflow Engine
11. `M10` Multi-Agent Collaboration

13. `M12` Open Source Quality and Public Release

## Immediate Objective

Finish `M0`, then build the first usable desktop flow:

1. lock architecture, docs, ADRs, and package boundaries
2. validate desktop shell, configuration, logging, database, and audit foundations
3. add plugin SDK and MCP gateway in `M2`
4. build GitHub plugin in `M3`
5. build the PR review workflow after connector and agent foundations exist
